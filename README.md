# My Streamlit App

This is your super researcher LLM.

## Quick Start

1. Clone this repository:

   ```
   git clone https://github.com/yourusername/my_streamlit_app.git
   cd my_streamlit_app
   ```

2. Run the setup script:

   ```
   chmod +x setup.sh
   ./setup.sh
   ```

3. Run the app:
   ```
   ./run_app.sh
   ```

The app will open in your default web browser.
